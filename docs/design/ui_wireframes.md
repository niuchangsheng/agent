# SECA UI 线框图与页面流转

## 1. 全局视觉基调
- **风格**: Hacker/极客深色模式 (Glassmorphism + Dark Mode)。
- **主色调**: 幽默灰蓝底色 (`#0f172a`)，关键状态或推演阶段使用荧光青 (`#06b6d4`) 及代码结构绿 (`#22c55e`) 高亮。
- **动效**: 提供平滑的代码展开微动画（Micro-animations），不突兀，强调可观测性的流动感。

## 3. 回放审计室 (Post-mortem Playback & Review)

## 4. 页面流转梳理
1. 用户在 **Homepage** 发起新的对话（提供简单的目标：如 "重压测试显示存在瓶颈"）。
2. UI 立即平滑切换入 **Introspection Dashboard**，流式输出诊断全景和推理进展。
3. 当 Agent 工作陷入停滞时，仪表盘亮起 `User Intervention Required` 红灯。
4. 任务不论成功完毕，界面显示 Summary，并允许点击特定分支进入 **Playback Room** 查看并核准代码推送。
