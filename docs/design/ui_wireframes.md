# SECA UI 线框图与页面流转

## 1. 整体基调与审美准则
- **色阶**: 黑客感冷深系，基础背景以碳灰/靛蓝色(`bg-slate-950`)为主。采用 Glassmorphism 毛玻璃特性（背挂浅荧光渐变晕染）以体现高度专业透明化的前卫工具感。
- **指示色**: 通过极光青 (`text-cyan-400`) 对比当前追踪事件；血红 (`border-red-500` 呼吸特效) 表示被 Harness 沙箱制止的灾难阻断情况。
